# Générateur de présentation Solimed

## Fichiers
- `app.py`
- `requirements.txt`
- `templates.pptx`

## Lancer en local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déployer sur Streamlit Cloud
1. Créer un repo GitHub.
2. Mettre ces 3 fichiers à la racine.
3. Aller sur Streamlit Cloud.
4. Déployer le repo en choisissant `app.py`.
